# Changelog

## 0.8.1-beta - 2026-05-05

Beta build for wider Streamer.bot testing.

- Added working provider setup and donation polling for DonatePay RU, DonatePay EU, DonateX.gg, Streamlabs, deStream, Donate.Stream, and Generic API.
- Added per-provider and `Any donation` Custom Code Event triggers.
- Added normalized donation variables and aliases for Streamer.bot actions.
- Added native Streamer.bot Credits donation integration.
- Added donation goal variables and reset/setup actions.
- Added donation marathon/timer variables and reset/setup actions.
- Added separate OBS overlays for donation goal and donation timer.
- Added configurable overlay comments, colors, bare numeric mode, and service list visibility.
- Added `!resetDC` command for broadcaster/moderator restart.
- Added compact diagnostics for provider polling state.
