# История изменений

Подробный патчноут Beta 3: [PATCH_NOTES_RU.md](PATCH_NOTES_RU.md)

## 0.13.0-beta.3.15 - DonConnect Beta 3 Hotfix 15

- Таймерный виджет больше не показывает строку последнего донатера: этот блок оставлен только для цели.
- Стартовая раскладка алерта доната выровнена ближе к центру, чтобы элементы не выглядели съехавшими при первом запуске.
- Очищены старые provider-значения из setup actions (`Client ID`, `Client Secret`, JWT, API keys, access tokens и custom endpoint), чтобы импортируемый `.sb` не содержал данные тестовой машины.
- Добавлен `BETA3_CLOUD_REFERENCE_RU.md` с полным референсом функций Beta 3 для облачной версии DonConnect.

## 0.13.0-beta.3.14 - DonConnect Beta 3 Hotfix 14

- Декоративная картинка таймера отделена от декоративной картинки цели: теперь reset и загрузка картинки в таймере не трогают оформление цели.
- Исправлены привязки слоя декора таймера в предпросмотре, включая перемещение, растягивание, поворот и сброс.
- Пересобран импортируемый `DonConnect.Beta3.sb`, чтобы внутри файла Streamer.bot был именно hotfix 14.

## 0.13.0-beta.3.13 - DonConnect Beta 3 Hotfix 13

- Логи донатов теперь пишутся в коротком формате: `dd/mm/yy - ник - сумма валюта - площадка - сообщение`.
- В OBS-доке кнопка открытия логов перенесена рядом с «Обновить» и называется «Открыть логи».
- Вкладка «Док OBS» в редакторе перенесена в конец списка, после «Запрет».
- Исправлено растягивание полоски и текста цели в предпросмотре, включая сброс выбранных элементов на значения по умолчанию.
- Исправлено выделение площадок и скрытых слоёв таймера; в отдельном таймере больше не показывается строка последнего доната.
- Добавлены декоративные картинки для алерта, таймера и лидерборда.
## 0.13.0-beta.3.12 - DonConnect Beta 3 Hotfix 12

- Исправлено растягивание слоёв в предпросмотре: увеличение одного элемента алерта, цели или таймера больше не должно сдвигать остальные редактируемые слои.
- В алерте стабилизированы слоты раскладки, поэтому увеличение донатера, суммы, площадки, сообщения или медиа не толкает соседние элементы.

## 0.13.0-beta.3.11 - DonConnect Beta 3 Hotfix 11

- Исправлен путь логов донатов: реальные записи по дням и кнопка открытия папки в редакторе/OBS-доке теперь смотрят в одну и ту же папку DonConnect.
- В прямом редакторе предпросмотра добавлено растягивание фона панели цели/таймера по высоте.
- Перед пересборкой Beta 3 перепроверены прямое редактирование элементов, автозапуск виджетов, экспорт/импорт профилей, отсутствие провайдерских секретов в профилях и встроенный JavaScript браузерных виджетов.

## 0.13.0-beta.3.10 - DonConnect Beta 3 Hotfix 10

- Усилен автозапуск локального сервера виджетов: после запуска Streamer.bot DonConnect теперь делает повторные попытки и включает watchdog, чтобы OBS-виджеты не оживали только после открытия редактора.
- Автосброс титров/лидерборда теперь запускается не только отдельным action, но и при старте DonConnect, с повторными попытками, пока Streamer.bot HTTP Server не будет готов.
- В OBS-док добавлена кнопка открытия папки логов донатов.
- На вкладке «Док OBS» в редакторе кнопка логов переименована в «Открыть папку логов по дням».

## 0.13.0-beta.3.9 - DonConnect Beta 3 Hotfix 9

- Action для старта стрима теперь называется `DonConnect - Stream Start Reset` и сбрасывает титры + лидерборд только один раз за локальный день.
- Повторный `Stream Online` в тот же день, например после отключения света или быстрого рестарта стрима, больше не очищает титры и лидерборд повторно.
- Ручной сброс титров стал строгим: если Streamer.bot HTTP Server не ответил на `/ClearCredits`, DonConnect показывает ошибку и не делает вид, что сброс выполнен.
- Настройка автоочистки лидерборда при запуске теперь тоже защищена от повторного срабатывания в тот же день.

## 0.13.0-beta.3.8 - DonConnect Beta 3 Hotfix 8

- В OBS-док добавлена маленькая кнопка «Сбросить титры»: она очищает титры DonConnect и, если доступен HTTP Server Streamer.bot, вызывает штатный сброс Credits.
- В пакет добавлен action `DonConnect - Reset Credits`, который можно повесить на `Twitch -> Channel -> Stream Online` для сброса титров при старте стрима.
- После ручного сброса титров DonConnect временно не подхватывает старый файловый кэш Credits, чтобы сброшенные титры не возвращались сразу обратно.
- Перепроверены критичные зоны перед Beta 3: автозапуск локального сервера, профили с медиа, логи донатов по дням, секреты провайдеров, обычный таймер, цель и редактор элементов.

## 0.13.0-beta.3 - DonConnect Beta 3

- Добавлен отдельный таймер сбора внутри виджета цели: он выключен по умолчанию, настраивается по дате/времени окончания, двигается в предпросмотре и имеет отдельный слой.
- Фильтрация титров Streamer.bot стала безопаснее: если схема настроек Streamer.bot не распознана, DonConnect больше не считает отсутствующие массивы секций выключенными секциями.
- Удалены встроенные данные приложения DonationAlerts: подключение теперь только через приложение, которое создаёт сам пользователь.
- Улучшен автозапуск: сервер редактора и OBS-виджетов поднимается независимо от ошибок запуска провайдеров.
- Страница подключения DonationAlerts получила понятную инструкцию, кнопку создания приложения и проверку Client ID/Client Secret перед авторизацией.
- Профили редактора теперь явно включают используемые медиа, но не включают токены площадок.
- Подтверждено состояние интеграций: Streamlabs, Donate.Stream, deStream и остальные существующие провайдеры оставлены активными; StreamEngine пока помечен как «В разработке».

## 0.12.1-beta.2.8 - DonConnect Beta 2 Hotfix 8

- Исправлена логика скорости титров: ползунок теперь работает как скорость, вправо значит быстрее.
- Титры DonConnect снова корректно показывают имена донатеров вместо `Viewer`, включая старые записи с полями `donor`/`user`.
- Длинные титры автоматически ускоряются, если выключена галочка «Не ускорять длинные титры».
- Улучшена подстановка установленных Windows-шрифтов и скачанных шрифтов с суффиксами `Regular/Bold/Italic`.
- Групповое перемещение элементов в предпросмотре больше не двигает вложенные элементы с двойной скоростью.
- Текст и медиа при ручном изменении размера больше не обрезаются рамкой алерта/цели.
- Анимация смены слайдов лидерборда стала мягче.
- В пакет импорта добавлен актуальный код Hotfix 8.

## 0.12.1-beta.2.7 - DonConnect Beta 2 Hotfix 7

- Страница подключения площадок стала компактной: все площадки сворачиваются, а в заголовке сразу показано состояние подключения.
- Статус площадки использует встроенную диагностику DonConnect и показывает ошибку соединения, если она есть.
- В редактор виджетов добавлена кнопка перехода к подключению площадок; ссылки DonatePay RU и EU ведут прямо на страницу API.
- Добавлен плейсхолдер StreamEngine «В разработке». Streamlabs, Donate.Stream и deStream оставлены доступными, поскольку их провайдеры уже реализованы.
- Для алерта, цели и таймера добавлены перемещение, изменение ширины и высоты, поворот и отдельный сброс выбранного элемента.
- В предпросмотр добавлены включаемая сетка, центральные оси холста и направляющие центра выбранного элемента без прилипания.
- Картинки алерта и цели можно перемещать, растягивать и поворачивать прямо в предпросмотре.
- Интерактивные рамки убраны из титров и лидерборда: эти разделы пока редактируются только обычными настройками.

## 0.12.1-beta.2.6 - DonConnect Beta 2 Hotfix 6

- Титры теперь фильтруются по включенным секциям в штатных настройках Credits Streamer.bot; отдельный выбор скорости удален, длительность рассчитывается по объему данных.
- Добавлено резервное JSON-хранилище настроек, токенов и текущего состояния цели/таймера в папке DonConnect рядом со Streamer.bot.
- Исправлено сохранение дробной прозрачности и добавлено выравнивание текста лидерборда.
- В алерт добавлены отдельные X/Y-настройки донатера, суммы, сообщения и площадки.
- Исправлена ручная резервная конвертация валют; тест таймера теперь использует тот же конвертер, что и реальные донаты.
- Добавлены экспорт и импорт профилей оформления без токенов провайдеров; используемые медиа алертов входят в профиль.
- Добавлена браузерная страница подключения площадок: сохраненные секреты не возвращаются обратно в браузер.
- В предпросмотр добавлено прямое перемещение и изменение размера поддерживаемых элементов.
- Подсказка с рекомендуемым размером Browser Source в OBS сделана крупнее и заметнее.

## 0.12.1-beta.2.5 - DonConnect Beta 2 Hotfix 5

- Исправлен тест титров: тестовые данные больше не заменяются обычным `credits.cache` через секунду.
- Кнопка теста теперь включает стабильный тестовый режим, а повторное нажатие возвращает реальные титры Streamer.bot.
- Титры Streamer.bot и донаты DonConnect теперь выводятся одновременно, а не заменяют друг друга.
- Управление секциями Streamer.bot удалено из редактора DonConnect: включение и отключение секций остаётся в штатных настройках Credits самого Streamer.bot.
- Для отдельной секции донатов DonConnect добавлены настройки заголовка и отображения имени, суммы, площадки и сообщения.
- Обычный режим больше не подставляет фальшивые донаты, если история донатов DonConnect пуста.

## 0.12.1-beta.2.4 - DonConnect Beta 2 Hotfix 4

- Исправлены сильные лаги титров из-за блокирующих запросов к HTTP Server Streamer.bot и полной перерисовки при каждом опросе.
- DonConnect теперь сначала читает локальный `data/credits.cache` Streamer.bot, а короткий HTTP-запрос использует только как запасной вариант в фоне.
- Возвращён полный список секций Streamer.bot Credits, включая пустые секции, роли, Hype Train, Bits, награды, группы и пользовательские секции.
- Исправлен возврат снятых галочек секций после сохранения или перезагрузки редактора.
- Сохранение настроек титров больше не перезапускает анимацию принудительно.

## 0.12.1-beta.2.3 - DonConnect Beta 2 Hotfix 3

- В разделе **Озвучка доната** добавлены отдельные галочки: зачитывать имя донатера, сумму, площадку и сообщение.
- Настройки озвучиваемых строк сохраняются в JSON и применяются к реальным донатам, тестовой кнопке голоса и запасному browser-TTS режиму.
- По умолчанию все строки включены, чтобы старое поведение не менялось само.

## 0.12.1-beta.2.2 - DonConnect Beta 2 Hotfix 2

- Срочно исправлена озвучка донат-алертов: DonConnect теперь корректно загружает `System.Speech` и дополнительно использует fallback через Windows SAPI.
- Кнопка проверки голоса теперь возвращает результат проверки и показывает движок озвучки или ошибку.
- Список голосов в редакторе теперь показывает серверные голоса Windows, которые реально может использовать DonConnect.
- Если выбранный ранее голос больше не найден, DonConnect пробует голос Windows по умолчанию вместо полной тишины.

## 0.12.1-beta.2 - DonConnect Beta 2

- Добавлены OBS-виджеты для донат-алерта, цели, таймера, титров, лидерборда и компактной OBS док-панели.
- Добавлены медиатека, повторы алертов, кастомный тестовый алерт, озвучка текста и фильтр запретных слов.

- Добавлена проверка новых версий при запуске через `version.json` на GitHub.
