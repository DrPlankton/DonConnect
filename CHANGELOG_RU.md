# История изменений

Подробный патчноут Beta 2: [PATCH_NOTES_RU.md](PATCH_NOTES_RU.md)

## 0.12.1-beta.2.1 - DonConnect Beta 2 Hotfix 1

- Исправлена озвучка донатов через голоса Windows, добавлены выбор голоса и кнопка проверки в редакторе.
- OBS-док получил мини-индикатор цели, кнопку сброса цели и мини-таймер.
- Исправлено ручное обновление OBS-дока.
- README обновлён простой инструкцией для новичков, подсказкой про **Test Trigger**, обновление поверх старой версии и ручное назначение `Ctrl+Alt+D`.

## 0.12.1-beta.2 - DonConnect Beta 2

- Добавлен встроенный браузерный редактор виджетов, который запускается самим расширением Streamer.bot.
- Добавлены OBS-виджеты для донат-алерта, цели, таймера, титров, лидерборда и компактной OBS док-панели.
- Добавлены медиатека, повторы алертов, кастомный тестовый алерт, озвучка текста и фильтр запретных слов.
- Расширены редакторы цели, таймера, титров и лидерборда: пресеты, live preview и подробные настройки внешнего вида.
- Настройки теперь сохраняются локально в JSON-файлы в папке `DonConnect` рядом со Streamer.bot.
- Добавлена проверка новых версий при запуске через `version.json` на GitHub.
- Добавлены заметки по ODA и предупреждения о возможных дублях при подключении агрегаторов и исходных площадок одновременно.
- Старые установочные файлы заменены одним файлом импорта Beta 2: `DonConnect.Beta2.sb`.
