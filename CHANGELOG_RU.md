# История изменений

Подробный патчноут Beta 2: [PATCH_NOTES_RU.md](PATCH_NOTES_RU.md)

## 0.12.1-beta.2.7 - DonConnect Beta 2 Hotfix 7

- Страница подключения площадок стала компактной: все площадки сворачиваются, а в заголовке сразу показано состояние подключения.
- Статус площадки использует встроенную диагностику DonConnect и показывает ошибку соединения, если она есть.
- В редактор виджетов добавлена кнопка перехода к подключению площадок; ссылки DonatePay RU и EU ведут прямо на страницу API.
- Добавлен плейсхолдер StreamEngine «В разработке». Streamlabs, Donate.Stream и deStream оставлены доступными, поскольку их провайдеры уже реализованы.
- Для алерта, цели и таймера добавлены перемещение, изменение ширины и высоты, поворот и отдельный сброс выбранного элемента.
- В предпросмотр добавлены включаемая сетка, центральные оси холста и направляющие центра выбранного элемента без прилипания.
- Картинки алерта и цели можно перемещать, растягивать и поворачивать прямо в предпросмотре.
- Интерактивные рамки убраны из титров и лидерборда: эти разделы пока редактируются только обычными настройками.

## 0.12.1-beta.2.6 - DonConnect Beta 2 Hotfix 6

- Титры теперь фильтруются по включенным секциям в штатных настройках Credits Streamer.bot; отдельный выбор скорости удален, длительность рассчитывается по объему данных.
- Добавлено резервное JSON-хранилище настроек, токенов и текущего состояния цели/таймера в папке DonConnect рядом со Streamer.bot.
- Исправлено сохранение дробной прозрачности и добавлено выравнивание текста лидерборда.
- В алерт добавлены отдельные X/Y-настройки донатера, суммы, сообщения и площадки.
- Исправлена ручная резервная конвертация валют; тест таймера теперь использует тот же конвертер, что и реальные донаты.
- Добавлены экспорт и импорт профилей оформления без токенов провайдеров и файлов медиатеки.
- Добавлена браузерная страница подключения площадок: сохраненные секреты не возвращаются обратно в браузер.
- В предпросмотр добавлено прямое перемещение и изменение размера поддерживаемых элементов.
- Подсказка с рекомендуемым размером Browser Source в OBS сделана крупнее и заметнее.

## 0.12.1-beta.2.5 - DonConnect Beta 2 Hotfix 5

- Исправлен тест титров: тестовые данные больше не заменяются обычным `credits.cache` через секунду.
- Кнопка теста теперь включает стабильный тестовый режим, а повторное нажатие возвращает реальные титры Streamer.bot.
- Титры Streamer.bot и донаты DonConnect теперь выводятся одновременно, а не заменяют друг друга.
- Управление секциями Streamer.bot удалено из редактора DonConnect: включение и отключение секций остаётся в штатных настройках Credits самого Streamer.bot.
- Для отдельной секции донатов DonConnect добавлены настройки заголовка и отображения имени, суммы, площадки и сообщения.
- Обычный режим больше не подставляет фальшивые донаты, если история донатов DonConnect пуста.

## 0.12.1-beta.2.4 - DonConnect Beta 2 Hotfix 4

- Исправлены сильные лаги титров из-за блокирующих запросов к HTTP Server Streamer.bot и полной перерисовки при каждом опросе.
- DonConnect теперь сначала читает локальный `data/credits.cache` Streamer.bot, а короткий HTTP-запрос использует только как запасной вариант в фоне.
- Возвращён полный список секций Streamer.bot Credits, включая пустые секции, роли, Hype Train, Bits, награды, группы и пользовательские секции.
- Исправлен возврат снятых галочек секций после сохранения или перезагрузки редактора.
- Сохранение настроек титров больше не перезапускает анимацию принудительно.

## 0.12.1-beta.2.3 - DonConnect Beta 2 Hotfix 3

- В разделе **Озвучка доната** добавлены отдельные галочки: зачитывать имя донатера, сумму, площадку и сообщение.
- Настройки озвучиваемых строк сохраняются в JSON и применяются к реальным донатам, тестовой кнопке голоса и запасному browser-TTS режиму.
- По умолчанию все строки включены, чтобы старое поведение не менялось само.

## 0.12.1-beta.2.2 - DonConnect Beta 2 Hotfix 2

- Срочно исправлена озвучка донат-алертов: DonConnect теперь корректно загружает `System.Speech` и дополнительно использует fallback через Windows SAPI.
- Кнопка проверки голоса теперь возвращает результат проверки и показывает движок озвучки или ошибку.
- Список голосов в редакторе теперь показывает серверные голоса Windows, которые реально может использовать DonConnect.
- Если выбранный ранее голос больше не найден, DonConnect пробует голос Windows по умолчанию вместо полной тишины.

## 0.12.1-beta.2.1 - DonConnect Beta 2 Hotfix 1

- Исправлена озвучка донатов через голоса Windows, добавлены выбор голоса и кнопка проверки в редакторе.
- OBS-док получил мини-индикатор цели, кнопку сброса цели и мини-таймер.
- Исправлено ручное обновление OBS-дока.
- README обновлён простой инструкцией для новичков, подсказкой про **Test Trigger**, обновление поверх старой версии и ручное назначение `Ctrl+Alt+D`.

## 0.12.1-beta.2 - DonConnect Beta 2

- Добавлен встроенный браузерный редактор виджетов, который запускается самим расширением Streamer.bot.
- Добавлены OBS-виджеты для донат-алерта, цели, таймера, титров, лидерборда и компактной OBS док-панели.
- Добавлены медиатека, повторы алертов, кастомный тестовый алерт, озвучка текста и фильтр запретных слов.
- Расширены редакторы цели, таймера, титров и лидерборда: пресеты, live preview и подробные настройки внешнего вида.
- Настройки теперь сохраняются локально в JSON-файлы в папке `DonConnect` рядом со Streamer.bot.
- Добавлена проверка новых версий при запуске через `version.json` на GitHub.
- Добавлены заметки по ODA и предупреждения о возможных дублях при подключении агрегаторов и исходных площадок одновременно.
- Старые установочные файлы заменены одним файлом импорта Beta 2: `DonConnect.Beta2.sb`.
